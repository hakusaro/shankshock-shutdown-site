It was inevitable that it would end like this. After several years of hosting Minecraft, ShankShock is ceasing operations as a Minecraft server host.

On February 4th, 2015, Mojang will enable name changes across the Minecraft login system. While newer servers will be able to cope with this, many older servers will not. The server API we used, Bukkit was all but abandoned after it turned out that Mojang bought the software, as well as the team behind it. This, followed by a DMCA takedown notice from a former Bukkit developer lead to the effective shutdown and discontinuation of the server mod that we use to build the ShankShock Core services and infrastructure. While projects like Spigot and Sponge are attempting to pick up the slack, no clear successor has defined itself as of late. To compound things, the primary method for generating income on ShankShock was traditionally a donation system that offered in game benefits to users in exchange for real money. A recent Mojang statement already declared this type of monetization against their terms of service, despite the fact that many servers (including ours) had fair, equal redemption systems in place designed to deter pay to win capabilities. As a result, for the last several months, the server has operated in a soft shutdown state. This state allowed the server to continue operating, but no updates nor features were added or developed. Now that name changes are being enabled, it is no longer safe to continue hosting the server. Any admin who changes their name would effectively allow impersonation of themselves on our server. Further, any donor or other privileged user would open up the same possibilities.

Donors with active forum accounts will be able to download the server within the coming weeks, including all plugins as well as database backups and the maps. The ShankShock Core is already freely available on GitHub. The forums will be removed entirely (along with server downloads) soon after.

The community lives on via Teamspeak, and many admins and donors can be found there. Thanks for being apart of something magical.


Lucas "Shank" Nicodemus
